📖 Evangelho do Dia
Um projeto simples e direto que exibe o Evangelho do Dia em uma página web, utilizando HTML, CSS e JavaScript.
O objetivo é oferecer uma interface limpa e acessível para acompanhar as leituras diárias do Evangelho.

✨ Funcionalidades
Exibição do Evangelho do Dia em formato web.

Interface estilizada com CSS personalizado.

Consumo de dados via API em JavaScript (API_Leituras.js).

Estrutura leve e fácil de modificar.

📂 Estrutura do Projeto
Arquivo/Folha	Função
index.html	Página principal que renderiza o conteúdo do Evangelho.
style_evangelho.css	Estilos visuais da aplicação (cores, fontes, layout).
API_Leituras.js	Script responsável por buscar e exibir as leituras do dia.
LICENSE	Licença MIT para uso e distribuição.
.gitattributes	Configurações de atributos do Git.
