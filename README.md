# 📖 Evangelho do Dia

Um projeto simples, direto e minimalista que exibe o Evangelho do Dia em uma página web, oferecendo uma interface limpa e acessível para acompanhar as leituras diárias.

👉 **Acesse o projeto:** [joaovitorreiss.github.io/Evangelho-do-dia](https://joaovitorreiss.github.io/Evangelho-do-dia/)

---

## 🛠️ Tecnologias e API
O projeto foi desenvolvido utilizando **HTML, CSS e JavaScript**. 

A base de dados do sistema é alimentada pela seguinte API de Liturgia Diária:
- **API Utilizada:** [liturgia.up.railway.app/v2](https://liturgia.up.railway.app/v2/)
- **Repositório da API:** [Dancrf/liturgia-diaria](https://github.com/Dancrf/liturgia-diaria)

## 🚀 Como Executar Localmente

Como o projeto utiliza módulos JavaScript (`type="module"`), é necessário rodar um servidor local para evitar erros de política de CORS no navegador.

1. Clone o repositório:
   ```bash
   git clone https://github.com/joaovitorreiss/Evangelho-do-dia.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd Evangelho-do-dia
   ```

3. Inicie um servidor local de sua preferência. Exemplos:
   - **VS Code:** Abra a pasta e utilize a extensão **Live Server**.
   - **Node.js (NPM):**
     ```bash
     npx serve
     ```
   - **Python:**
     ```bash
     python -m http.server
     ```

4. Abra o link gerado pelo servidor no seu navegador (ex: `http://localhost:3000`).
